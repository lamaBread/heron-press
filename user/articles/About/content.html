About This Demo

This is a live demo site for Heron, a lightweight, PHP-targeted static site generator. Every article under user/articles/ is here to exercise a different feature, so the repository doubles as runnable documentation.

This particular page is written as content.html (a raw HTML body) rather than Markdown. HTML bodies are emitted verbatim, which makes them the right choice when you are migrating existing HTML or need exact control over the markup. Because nothing is auto-wrapped, an HTML body must supply its own <div class='gap'> + <section> structure by hand — view source to see the required idiom.
