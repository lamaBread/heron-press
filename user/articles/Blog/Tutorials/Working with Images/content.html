Working with Images

This tutorial lives in the Blog/Tutorials subcategory, so it also demonstrates how categories nest. It is an HTML body because it uses both image-box helpers.

Single image with a caption

The diagram below is a source PNG. At build time Heron downsizes it into the widths configured in site.yaml (400 / 800 / 1600), writes WebP variants, and emits an <img> with srcset, sizes, and loading="lazy". View source to see the generated markup.

A slideshow

imgSlideBox() turns a folder of images into a slideshow. All images are rendered server-side as <img class="slide">; the bundled imgslidebox.js only toggles which one is active, so the page still works without JavaScript (you simply see the first slide).

Both helpers are expanded to static HTML at build, so this page is emitted as index.html despite being authored with PHP calls.
