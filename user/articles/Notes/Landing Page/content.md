# A Standalone Landing Page

This page sets `use_common_css: false`, so the shared site stylesheet is not
linked at all — only this article's own `landing.css` styles it. That makes it
suitable for a focused landing or single-service page that should not inherit
the site chrome's look.

Everything here is still a normal Heron article: it has a slug, a date, SEO
metadata, and it appears in the Notes category and the feeds.
